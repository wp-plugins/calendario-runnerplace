Utiliza el calendario de carreras populares de RunnerPlace para tenerlo actualizado en tu web.

== Description ==
Ten el calendario de carreras populares (running, duatlones, triatlones, acuatlones, trails) de España en tu web.

Utiliza la API de [RunnerPlace][http://www.runnerplace.com] para que tus usuarios puedan buscar las carreras populares desde tu propia página web.

Filtros por Provincia, Tipo de Carrera y Mes.

== Installation ==

Esta sección describe cómo se instala el widget y cómo hacerlo funcionar

1. Sube el contenido de la carpeta `runnerplace-calendar` al directorio `/wp-content/plugins/` 
2. Activa  el plugin a través del menú 'Plugins' de WordPress
3. Añade el widget a la barra lateral
4. Selecciona (si quieres) algún deporte o provincia para que se muestre por defecto
5. Ayúdanos a expandir RunnerPlace añadiendo los enlaces a las fichas de carreras y el link poweredby RunnerPlace.
